📊 Iris & Tips Data Visualization App

📊 A Streamlit-based interactive data visualization project using Python, Pandas, Seaborn, and Matplotlib to explore and analyze the Iris and Tips datasets in a simple and visual way.

---

## 🚀 Overview

This project provides an easy-to-use web app for performing **Exploratory Data Analysis (EDA)** on two popular datasets: Iris and Tips. It helps users understand data patterns through interactive graphs and visual insights.

---

## ✨ Features

- 🌸 Iris dataset visualization (sepal & petal analysis, species distribution)
- 🍽️ Tips dataset visualization (total bill, tip, gender, smoker analysis)
- 📊 Count plots, histograms, and distribution plots
- 🎛️ Interactive Streamlit interface
- ⚡ Fast and beginner-friendly dashboard

---
